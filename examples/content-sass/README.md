# Sass Content

What you’ll see: a small badge injected into the page, styled with Sass.

How it works here: the content script uses Sass to organize styles, making the injected UI easy to maintain and extend.

## Installation

```bash
npx extension@latest create <project-name> --template content-sass
cd <project-name>
npm install
```

## Commands

### dev

Run the extension in development mode.

```bash
npm run dev
```

### build

Build the extension for production.

```bash
npm run build
```

### preview

Preview the extension in the browser.

```bash
npm run preview
```
