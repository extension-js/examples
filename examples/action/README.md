# JavaScript Action

What you’ll see: a toolbar popup with a simple page.

How it works here: the extension registers an action and loads `action/index.html` as the popup.

## Installation

```bash
npx extension@latest create <project-name> --template action
cd <project-name>
npm install
```

## Commands

### dev

Run the extension in development mode.

```bash
npm run dev
```

### build

Build the extension for production.

```bash
npm run build
```

### preview

Preview the extension in the browser.

```bash
npm run preview
```
