# Content Sass Modules Example

> Content scripts with Sass Modules.

Scoped Sass modules for content UIs. Compose complex styles without global leaks.

## Installation

```bash
npx extension@latest create <project-name> --template content-sass-modules
cd <project-name>
npm install
```

## Commands

### dev

Run the extension in development mode.

```bash
npm run dev
```

### build

Build the extension for production.

```bash
npm run build
```

### preview

Preview the extension in the browser.

```bash
npm run preview
```
