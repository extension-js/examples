# Content Example

> Modern content script example with clean injection patterns.

Shows how to inject scripts into web pages with isolated styles and maintainable code. Use this as a base for page augmentations.

## Installation

```bash
npx extension@latest create <project-name> --template content
cd <project-name>
npm install
```

## Commands

### dev

Run the extension in development mode.

```bash
npm run dev
```

### build

Build the extension for production.

```bash
npm run build
```

### preview

Preview the extension in the browser.

```bash
npm run preview
```
