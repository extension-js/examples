# React Router New Tab

What you’ll see: a custom new tab page with multiple routes.

How it works here: the extension overrides the new tab and loads a React app wired with React Router to handle multiple pages.

## Installation

```bash
npx extension@latest create <project-name> --template new-react-router
cd <project-name>
npm install
```

## Commands

### dev

Run the extension in development mode.

```bash
npm run dev
```

### build

Build the extension for production.

```bash
npm run build
```

### preview

Preview the extension in the browser.

```bash
npm run preview
```
