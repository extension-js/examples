# New Example

> Baseline starter for a new extension.

A clean foundation with sensible defaults, fast dev server, and production builds.

## Installation

```bash
npx extension@latest create <project-name> --template new
cd <project-name>
npm install
```

## Commands

### dev

Run the extension in development mode.

```bash
npm run dev
```

### build

Build the extension for production.

```bash
npm run build
```

### preview

Preview the extension in the browser.

```bash
npm run preview
```
