# Content TypeScript Example

> TypeScript-based content scripts.

Typed content logic and UI with fast builds and maintainable code.

## Installation

```bash
npx extension@latest create <project-name> --template content-typescript
cd <project-name>
npm install
```

## Commands

### dev

Run the extension in development mode.

```bash
npm run dev
```

### build

Build the extension for production.

```bash
npm run build
```

### preview

Preview the extension in the browser.

```bash
npm run preview
```
