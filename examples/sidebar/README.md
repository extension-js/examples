# Sidebar Example

> Sidebar (side panel) example.

Implements a side panel UI with routing, messaging, and persistent state.

## Installation

```bash
npx extension@latest create <project-name> --template sidebar
cd <project-name>
npm install
```

## Commands

### dev

Run the extension in development mode.

```bash
npm run dev
```

### build

Build the extension for production.

```bash
npm run build
```

### preview

Preview the extension in the browser.

```bash
npm run preview
```
