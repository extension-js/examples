# New Less Example

> Starter styled with Less.

Less preprocessor configured out of the box for rapid styling.

## Installation

```bash
npx extension@latest create <project-name> --template new-less
cd <project-name>
npm install
```

## Commands

### dev

Run the extension in development mode.

```bash
npm run dev
```

### build

Build the extension for production.

```bash
npm run build
```

### preview

Preview the extension in the browser.

```bash
npm run preview
```
