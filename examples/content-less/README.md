# Less Content

What you’ll see: a small badge injected into the page, styled with Less.

How it works here: the content script organizes styles with Less, making it easy to structure and adjust styling for injected UI.

## Installation

```bash
npx extension@latest create <project-name> --template content-less
cd <project-name>
npm install
```

## Commands

### dev

Run the extension in development mode.

```bash
npm run dev
```

### build

Build the extension for production.

```bash
npm run build
```

### preview

Preview the extension in the browser.

```bash
npm run preview
```
