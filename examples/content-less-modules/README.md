# Content Less Modules Example

> Content scripts with Less Modules.

Scoped Less modules for content UIs. Keep styles modular and conflict-free.

## Installation

```bash
npx extension@latest create <project-name> --template content-less-modules
cd <project-name>
npm install
```

## Commands

### dev

Run the extension in development mode.

```bash
npm run dev
```

### build

Build the extension for production.

```bash
npm run build
```

### preview

Preview the extension in the browser.

```bash
npm run preview
```
