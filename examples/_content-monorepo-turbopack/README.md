# Monorepo Turbopack Example

> Monorepo using Turbopack.

Demonstrates a multi-package setup with fast builds and shared tooling via Turbopack.

## Installation

```bash
npx extension@latest create <project-name> --template monorepo-turbopack
cd <project-name>
npm install
```

## Commands

### dev

Run the extension in development mode.

```bash
npm run dev
```

### build

Build the extension for production.

```bash
npm run build
```

### preview

Preview the extension in the browser.

```bash
npm run preview
```
