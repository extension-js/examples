# New Crypto Example

> Starter using Web Crypto APIs.

Demonstrates secure primitives and best practices when working with browser crypto APIs.

## Installation

```bash
npx extension@latest create <project-name> --template new-crypto
cd <project-name>
npm install
```

## Commands

### dev

Run the extension in development mode.

```bash
npm run dev
```

### build

Build the extension for production.

```bash
npm run build
```

### preview

Preview the extension in the browser.

```bash
npm run preview
```
