# Transformers.js Sidebar

What you’ll see: a browser sidebar (side panel) that runs a small Transformers.js demo.

How it works here: the extension registers a sidebar panel and loads a page that initializes a Transformers.js model sample.

## Installation

```bash
npx extension@latest create <project-name> --template transformers-js
cd <project-name>
npm install
```

## Commands

### dev

Run the extension in development mode.

```bash
npm run dev
```

### build

Build the extension for production.

```bash
npm run build
```

### preview

Preview the extension in the browser.

```bash
npm run preview
```
