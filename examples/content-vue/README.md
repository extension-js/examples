# Content Vue Example

> Content UI powered by Vue.

Vue-driven content UI with hot reload and production-ready output.

## Installation

```bash
npx extension@latest create <project-name> --template content-vue
cd <project-name>
npm install
```

## Commands

### dev

Run the extension in development mode.

```bash
npm run dev
```

### build

Build the extension for production.

```bash
npm run build
```

### preview

Preview the extension in the browser.

```bash
npm run preview
```
